## Aagedal Media Player

Based on the same fast engine as (Aagedal Media Converter)[https://github.com/aagedal/Aagedal-Media-Converter], but now for quickly just checking playback of files.

(As of 2026-02-20 this app supports a few more video formats than IINA (APV and VVC) and launches faster.)

### Features
- Playback every audio and video file in existence, through a combination of mpv and Apple native playback engines.
- Simple and fast UI.
- Display timecode of your source files.
- Use JKL playback
- Full screen player with quick cursor hide area (right side)
